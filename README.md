* 第一組
  * 分類：財經
  * 名稱：FinTank
  * 簡介：
      * 利用課程所學習之各種技術工具，開發一個個人金融服務的入口平台
        finTank(金融坦克車)平台開發策略:
         1. 創造一個客戶為中心的對話機器人
         2. 藉由網路爬蟲等技術，提供最有用的資訊
         3. 利用各種先進技術，包括機器學習等，長期優化提供資訊的品質
      * 內容涵蓋銀行服務、換匯服務、股票分析、基金資訊、期貨操作五大主題
      * 使用深度學習提供客戶人臉與情緒辨識，商品標價照相換匯計算，大廳保安警示，
      * 股票K線型態辨認，基金投資建議，台指期貨波動幅度推論等等服務功能。
